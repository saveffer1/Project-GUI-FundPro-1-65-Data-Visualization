Before running the script, you need to install the following packages:
    - pip install -r requirements.txt
      * or run pip install customtkinter==4.6.3 darkdetect==0.7.1 tkintermapview==1.17 matplotlib==3.6.2 Pillow==9.3.0 pandas==1.5.1 openpyxl==3.0.10
    - install the font in config folder "NotoSerifThai.ttf"
    - run the script: python main.py

Font
    Noto Serif Thai: NotoSerifThai.ttf
        https://fonts.google.com/noto/specimen/Noto+Serif+Thai

Image icon
    All image and icon: Flaticon
        https://www.flaticon.com/

Excel data
    All data: Excel file
        https://data.go.th/
    Data: ข้อมูลปริมาณการผลิตข้าวนาปรัง (ผลผลิต) ปี 2557+
        https://data.go.th/dataset/oae0023_2557

Python Libraries
    CustomTkinter: v4.6.3 for GUI
        https://github.com/TomSchimansky/CustomTkinter/wiki/
        https://pypi.org/project/customtkinter/
    TkinterMapView: v1.17 for map
        https://github.com/TomSchimansky/TkinterMapView
        https://pypi.org/project/tkintermapview/
    Matplotlib: v3.6.2 for chart
        https://matplotlib.org/
        https://pypi.org/project/matplotlib/
    Pandas: v1.5.1 for Excel data
        https://pandas.pydata.org/
        https://pypi.org/project/pandas/
    Pillow: v9.3.0 for image and icon
        https://pillow.readthedocs.io/en/stable/
        https://pypi.org/project/Pillow/
    Darkdetect: v0.7.1 for check dark mode
        https://github.com/albertosottile/darkdetect
